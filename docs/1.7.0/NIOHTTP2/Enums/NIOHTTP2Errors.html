NIOHTTP2Errors

public enum NIOHTTP2Errors

Errors that NIO raises when handling HTTP/2 connections.

  • ExcessiveOutboundFrameBuffering

    The outbound frame buffers have become filled, and it is not possible to buffer further outbound frames. This occurs when the remote peer is generating work faster than they are consuming the result. Additional buffering runs the risk of memory exhaustion.
